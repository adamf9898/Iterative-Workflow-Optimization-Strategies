## How to Use This Template

### Before Iteration
1. Fill in Iteration Information
2. Define Objectives and Success Criteria
3. Plan Capacity
4. Select and detail work items
5. Review with team

### During Iteration
1. Update Daily Task Tracking each day
2. Log progress on individual tasks
3. Document blockers immediately
4. Record learnings and decisions
5. Adjust plans as needed

### After Iteration
1. Conduct Iteration Review
2. Facilitate Retrospective
3. Calculate and analyze metrics
4. Document lessons learned
5. Plan next iteration
6. Archive for future reference

### Continuous
1. Review and update template based on team feedback
2. Adapt sections to fit your team's needs
3. Keep template simple and useful
4. Share insights with broader organization
